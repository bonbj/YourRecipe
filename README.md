# Your Recipe
Portal de Receitas do seu dia-a-dia.

##### Depedencies
1. Axios ^0.19.2
2. React ^16.13.1
3. React-dom ^16.13.1
4. React-scripts 3.4.1

##### Clone the project and install the dependencies:

```
$ git clone https://github.com/bonbj/YourRecipe.git
$ cd YourRecipe
$ npm install
$ npm start
```
